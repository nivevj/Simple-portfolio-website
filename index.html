Major Works

1. N-BLOG - A BLOG APPLICATION | Django

A blog application developed with Django that uses SQLite as the database. This blog application has different features as, Login and Signup for users, Publish blogs (allows creating a blog with rich text editor), Edit published blogs, Delete published blogs, Upvote or Downvote blogs posted by other users, Leave comments on other user blogs, Create a new category and publish blogs on it, Category-wise view of blogs published on the website, User can maintain a profile page with profile pic, bio and links to their socials also Update their profile page details.

2. RETAIL STORE STOCK INVENTORY ANALYTICS | Data Science

A digitized inventory management system that performs retail store stock analysis to achieve less inventory on hand and more stocks on sale with less manual labor. This addresses the problem of managing the retail store inventory using the time series forecasting algorithm. Out of the different models available in this algorithm, the SARIMA (Seasonal Autoregressive Integrated Moving Average) model is preferred as it outperforms the others with its ability to forecast future needs with past seasonal time series data. The newsvendor formula is used in the proposed SARIMA model to calculate the optimal order quantity for the inventory. Overall the model predicts the inventory needed to fulfill future customer orders over a period of time with appropriate datasets from the retail store.

4. FOODIE - A CALORIE TRACKER | Django

A web application that shows the calorie chart of food and the exercise suggestion to burn those calories. It also shows the alert message for food with high amount of sugar. Developed using Django and food calorie API from API Ninjas.
